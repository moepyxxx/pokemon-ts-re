# URL定義
URL定義のルールは下記です。

|  パス  |  説明  |
| ---- | ---- |
|  /  |  トップページ  |
|  /news  |  ニュース一覧（Contentful）  |
|  /news/:id  |  ニュース詳細（Contentful）  |
|  /all  |  ポケモン図鑑完成形  |
|  /bookpokemon  |  ポケモン図鑑一覧  |
|  /bookpokemon/:id  |  ポケモン図鑑詳細  |
|  /search  |  ポケモンを探す  |
|  /search/:id  |  ポケモンを探す場所  |
